Introducción

Métodos numéricos para encontrar raíces de ecuaciones no lineales. Estos algoritmos permiten resolver f(x) = 0 cuando no existen soluciones analíticas.

Método de Bisección

Divide repetidamente un intervalo por la mitad. Convergencia garantizada.

Usar Método
Regla Falsa

Interpolación lineal para encontrar raíces. Más eficiente que bisección.

Usar Método
Punto Fijo

Transforma f(x) = 0 en x = g(x). Convergencia condicional.

Usar Método
Newton-Raphson

Usa la derivada para convergencia rápida. Requiere calcular f'(x).

Usar Método
Método de la Secante

Aproxima la derivada. No requiere calcular f'(x) directamente.

Usar Método
Raíces Múltiples

Newton-Raphson modificado para raíces con multiplicidad > 1.
